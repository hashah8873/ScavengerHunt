# Scavenger Hunt App

## Overview

Scavenger Hunt App is an iOS application developed using SwiftUI for Assignment 3.

The app allows users to participate in a local business scavenger hunt by finding different items and marking them as completed. Users can track their progress and receive reward messages based on the number of items found.

## Features

- Display a list of scavenger hunt items
- Mark items as found using toggle controls
- Track progress with a progress bar
- View the number of completed items
- Receive reward messages based on progress
- Built using SwiftUI

## Technologies Used

- Swift
- SwiftUI
- Xcode

## How to Run

1. Open the project in Xcode.
2. Select an iPhone Simulator.
3. Click Run.
4. Explore the scavenger hunt and mark items as found.
