# Scavenger Hunt App

## Overview

Scavenger Hunt App is an iOS application developed using SwiftUI as part of Assignment 3.

The application allows users to participate in a local business scavenger hunt by finding different items and marking them as completed. Users can track their progress through a progress bar and receive reward messages based on the number of items found.

## Features

* Display a list of scavenger hunt items
* Mark items as found using toggle controls
* Track progress with a progress bar
* View the number of completed items
* Display reward messages based on progress
* Reset all completed items
* Show a congratulatory alert when all items are found
* Built using SwiftUI

## Technologies Used

* Swift
* SwiftUI
* Xcode

## How to Run

1. Open the project in Xcode.
2. Select an iPhone Simulator.
3. Click Run.
4. Mark items as found using the toggle controls.
5. Track progress and unlock rewards.
